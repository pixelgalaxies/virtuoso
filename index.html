Virtuoso

A grid library built with the CSS grid module.

Another Grid Library?

Yes! This one is built with the CSS grid module though. It's small, fast, and easy to use. There are no bloat classes and no fighting with the library. It is JUST a grid system.

Every row has a preset number of fractional units that its children elements will use to set their widths. For example, the element with the class name "md-4-8" will should expect two children. The first will be one fractional unit (1/3 of the page) and the second element will be 2 fractional units (2/3 of the page).

The letters at the beginning of the classname indicate at what point should the elements break into full width columns.
